# Lottie Animations Folder

This folder contains Lottie animation files (.json) for the onboarding screens.

## Required Files

You need to add **5 Lottie animation files** here:

1. `welcome.json` - Welcome/intro animation
2. `flashcards.json` - Flashcard flip animation
3. `ai_solver.json` - AI problem solver animation
4. `gamification.json` - Gamification/achievement animation
5. `learning_path.json` - Learning path/roadmap animation

## Where to Download

See the comprehensive guide: **`LOTTIE_ANIMATIONS_GUIDE.md`** in the mobile folder

Quick link: [LottieFiles.com](https://lottiefiles.com/)

## Note

The app will work without these animations (fallback UI is provided), but animations make the onboarding experience much more engaging!

## File Requirements

- Format: Lottie JSON (.json)
- Size: < 100KB per file recommended
- Duration: 2-4 seconds loop
- Style: Modern, clean, matches emerald green theme
